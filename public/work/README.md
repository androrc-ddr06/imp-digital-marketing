# Swapping in your real screenshots

The images in this folder are **branded placeholders**. To show your real work
(Instagram growth screenshots, the brand boards, website previews), just replace
the file with your own image **using the same filename**. No code changes needed.

| Replace this file        | With a screenshot of…                              |
| ------------------------ | -------------------------------------------------- |
| `roosters.svg`           | Roosters Rolling BBQ — best brand/social shot      |
| `mjl.svg`                | MJL Plumbing rebrand                                |
| `lillard.svg`            | Lillard Restoration Services rebrand               |
| `site-roosters.svg`      | roostersrollingbbq.com homepage                    |
| `site-the10.svg`         | the10futbol.com homepage                           |
| `site-ccca.svg`          | cccaworks.org homepage                             |
| `site-camaron.svg`       | elcamaroncrudo.com homepage                         |

Tips:
- You can use `.jpg`, `.png`, or `.webp` — if you do, update the matching `image:`
  path in `src/lib/content.ts` (e.g. change `roosters.svg` to `roosters.jpg`).
- Best aspect ratio: roughly **4:3** for case studies, **16:11** for website previews.
- Keep files reasonably sized (under ~500 KB) for fast loading.
