# Project images

Most images here are **real client work**. A few are still branded placeholders —
to swap one, just replace the file with your own image **using the same filename**
(no code changes needed). To change a filename or aspect, update the matching path
in `src/lib/content.ts`.

Tips:
- `.jpg`, `.png`, or `.webp` all work. If you change the extension, update the
  matching `image:` / `src:` path in `src/lib/content.ts`.
- Best aspect ratios: ~4:3 for logos/case studies, ~16:10 for mockups, ~16:11 for
  website previews. Keep files under ~500 KB for fast loading.
